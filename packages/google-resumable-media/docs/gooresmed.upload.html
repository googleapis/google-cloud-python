gooresmed.upload module¶

Support for resumable uploads.

Also supported here are simple (media) uploads and multipart uploads that contain both metadata and a small file as payload.

class gooresmed.upload.Upload¶

Bases: object

Helper to manage uploading some a resource to a Google API.

Basic support will

  • upload directly from a file on the host OS
  • upload from a stream or file-like object
  • upload content directly from a string or bytes

For large files / content, the upload process will need to be split into smaller chunks as a resumable upload.
